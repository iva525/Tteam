# MiniSteam (demo)

This folder contains a lightweight "MiniSteam" demo — a mock library UI and several small browser demo games and utilities.

How to run locally:
- Open `mini-steam/index.html` in your browser (static).
- Or run the bundled Flask server in `mini-steam/server.py`:
  - pip install -r mini-steam/requirements.txt
  - python mini-steam/server.py

I added demos: Pong (canvas), Clicker (DOM) and placeholders for Tetris/Runner. There are also utilities: analytics, preloader, modal, localstorage extension and thumbnails.
